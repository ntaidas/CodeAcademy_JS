Skirtingose svetainesa ir skirtingose narsyklese Cookies, Session Storage ir Local Storage yra saugojama atskirai.

Cookies

Duomenys saugojami svetaineje.

Saugojama iki 4kb.
Reikia rankiniu budu nustatyti iki kada duomenys saugojami.
Siunciami kartu requestais i Back-End.

Session Storage

Laiko duomenis aplikacijoje, per kuria atidaroma kazkuri svetaineje, kuriai tie duomenys priskiriami.
Kai pasibaigia sesija (kai yra isjungiama svetaine) sitie duomenys yra istrinimai.

  • sessionStorage.setItem('raktas','reiksme')
  • sessionStorage.getItem('raktas')
  • sessionStorage.removeItem('raktas')
  • sessionStorage.clear()
  • sessionStorage.key(index)
  • sessionStorage.length

Local Storage

Laiko duomenis aplikacijoje, per kuria atidaroma kazkuri svetaineje, kuriai tie duomenys priskiriami.
Kai pasibaigia sesija (kai yra isjungiama svetaine) sitie duomenys yra issaugoti. Istrinami gali buti tik naudojant atitinkamus metodus arba per dev tools.

  • localStorage.setItem('raktas','reiksme')
  • localStorage.getItem('raktas')
  • localStorage.removeItem('raktas')
  • localStorage.clear()
  • localStorage.key(index)
  • localStorage.length

Java Script Object Notation (JSON)

JSON yra JS objektu aprasymas stringu pavidalu.
JSON viduje viskas rasoma kaip kad butu rasomas objektas,BET:
1) JSON viduje negali buti metodu. Jeigu metodas yra, jis yra panaikinamas.
2) JSON viduje visi zodziai apart skaiciu ir boolean turi buti stringai su dvigubomis kabutemis.

Objekta galima paversti i JSON sintakse arba JSON sintakse galima paversti i objekta naudojant atitinkamus metodus.

JSON Metodai

  • JSON.stringify(objektas)
  • JSON.parse(JSON_stringas)
