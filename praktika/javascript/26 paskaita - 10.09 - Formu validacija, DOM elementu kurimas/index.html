HTML formu validavimas

Formu validavimas yra reikalinas norint nustatyti kazkokius reikalavimus formos laukams(kitaip pasakius, inputu reikalavimus).
Formu validacija neleidzia vartotojui pateikti formos, jeigu bent vienas is formos lauku neatitinka nurodytos validacijos

  • required
  • minlength ir maxlength - text/password/url/email
  • min ir max - number/data/range
  • step - number/rage
  • pattern - password/tel
  • accept - file
  • multiple - file/email

Node'as - HTML elementas JS aplinkoje

Daug geresnis budas kurti HTML elementus per JS yra n uzpildant tevinio elemento innerHTML'a ar textContent, bet kuriant Nodes kaip kintamuosius ir priskiriant juos vieni kitiems.

Node metodai

  • createElement('TagVardas')
  • createTextNode('tekstas...')
  • kam.append(ka,kaDar,kaDar,...) - galima bet ka ir bet kiek
  • kam.appendChild(ka) - galima TIK node elementa
  • kam.setAttribute('name','value')
  • classList
    • add('klasesVardas','kitaKlase',...)
    • remove('vienaKlase','kitaKlase',...)
    • replace('klasesVardas', 'naujaKlase')
    • toggle('klasesVardas')
