Libraries / Frameworks

CSS Framewwork/Libraries - kazkieno kito parasytas CSS kodas, kuri tu gali naudoti savo projektuose. Norint naudotis frameworku, reikia ji prijungti prie savo kodo naudojant CDN. Dauguma frameworku veikia suteikiant ju aprasytas klases kazkokiems html elementams. Yra Classless CSS frameworku. Juos naudojant, nereikia suteikti HTML elementams klasiu. Viskas aprasyta naudojant tag'u/atributu/nth... selektorius.

CSS Frameworks

  • Bootstrap
  • Tailwind

  • UI Kit

Pros

  • Nereikia/labai mazai reikia rasyti CSS
  • Pagreitina/palengvina darba
  • lengva naudotis
  • Suprogramuoti responsiveness

Cons

  • Daug perteklinio/ nenaudojamo kodo
  • Reikia skirti laiko ismokti
  • Neunikalus dizainas
  • Prastesnis google paieskos sistemos (SEO) ivertinimas

Preprocessors

CSS with superpowers

Suteikia galimybe rasyti CSS naujoviskesniu budu, kur galima lengvai kurti ir naudoti:

  • Kintamuosius
  • nestinti selektorius
  • Galima rasyti:
    • Salygas
    • Ciklus
    • Funkcijas
    • mixinus
    • ir t.t...

Naudojant kazkuri CSS preprocesoriu, reikia parasyta koda konvertuoti i paprasta CSS, kuri supranta svetaine.

Preprocesoriu sarasas

  • SASS/SCSS
  • LESS
  • Stylus
  • PostCSS
  • ...

Measurment units

  • px
  • % - nustato pagal tevinio elemento dydi
  • vh/vw - viewport height, viewport width
  • rem - 1rem === 16px (is root nustatymo)
  • em - artimiausio protevinio elemento dydis
