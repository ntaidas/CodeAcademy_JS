Display Flex

  • Parent(container)
    • display: flex - suteikia konteineriui flex display
    • flex-direction: row/row-reverse/column/column-reverse - nurodo kuria kryptimi isdestomi elementai flex conteinerio viduje
    • flex-wrap: nowrap/wrap/wrap-reverse - nurodo ar/kaip elementai netelpantys konteinerio viduje pereis i kitas eilutes/stulpelius
    • flex-flow: - flex-direction su flex-wrap
    • justify-content: flex-start/flex-end/center/space-between/space-around/space-evenly - nurodo, kaip elementai issidesto main asyje
    • align-content: stretch/flex-start/flex-end/center/space-between/space-around/space-evenly - nurodo kaip elementai(visi bendrai) issidesto cross asyje
    • align-items: stretch/flex-start/flex-end/center/baseline - nurodo kaip elementai(atskiros eilutes/stulpeliai) issidesto cross asyje.
    • gap/column-gap/row-gap: nurodo tarpa tarp elementu conteineryje
  • Child(container items)
    • order: number - nurodo vaikinio elemento eiliskumo lygmeni
    • flex-grow: number - nurodo elemento didejimo santyki
    • flex-shrink: number - nurodo elemento mazejimo(jei tenka mazeti) santyki
    • flex-basis: %/px/rem/number - nurodoma elemento viso galimo plocio(kalbant apie tevinio elemento ploti) dalis
    • flex: flex-grow|flex-shrink|flex-basis
    • align-self: stretch/flex-start/flex-end/center/baseline - nurodo kaip pavienis vaikinis elementas turi atsistoti cross asyje(veikia su align-items, ne su align-content)
