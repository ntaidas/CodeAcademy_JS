Grid

  • Parent(container)
    • display:grid|inline-grid; -
    • grid-template-columns: unit/s ; - nurodai kiek ir kokio dydzio stulpeliu nori savo kuriamame tinklelyje
    • grid-template-rows: unit/s ; - nurodai kiek ir kokio dydzio eiluciu nori savo kuriamame tinklelyje
    • grid-template: grid-template-rows / grid-template-columns ;
    • grid-template-areas: "name | nameOtherOrSame | more | . (tuscias) | none(undefined) | ..." ;
    • row-gap: unit; nurodo tarpus tarp eiluciu
    • column-gap: unit; nurodo tarpus tarp stulpeliu
    • gap: row-gap column-gap;
    • align-items: ... ; nusako kur elementus pastatyti jiems skirto tinklelio viduje, Y asyje
    • justify-items: ... ; nusako kur elementus pastatyti jiems skirto tinklelio viduje, X asyje
    • place-items: align-items justify-items;
    • align-content: ...; nusako kaip visas gridas issidesto konteinerio viduje Y asyje
    • justify-content: nusako kaip visas gridas issidesto konteinerio viduje X asyje
    • place-content: align-content justify-content;
    • grid-auto-columns: ... ; - veikia keistai, tam kad pastatyti elementus, dar neegzistuojancioj vietoj(naudojamas tik isimtiniais atvejais)
    • grid-auto-rows: ... ; - veikia kaip grid-template-rows, tik tuo paciu metu nurodo kaip eilutes kurtusi, jei ju butu daugiau nei nurodyta.
    • grid-auto-flow: row | column | dense | row dense | column dense; - nusako kaip elementai issidesto gride, jeigu jiems nera priskirtos vietos
    • grid: grid-template-rows / grid-template-columns / grid-template-areas / grid-auto-rows / grid-auto-columns / grid-auto-flow
  • Child(elements in container)
    • grid-column-start: number | spanNumber;
    • grid-column-end: number | spanNumber;
    • grid-row-start: number | spanNumber;
    • grid-row-end: number | spanNumber;
    • grid-row: grid-row-start grid-row-end;
    • grid-column: grid-column-start grid-column-end;
    • grid-area: grid-row-start / grid-column-start/ grid-row-end / grid-column-end;
    • align-self: ... ; tas pats kaip ir flexe
    • justify-self: ... ; tas pats kaip ir flexe
    • place-self: align-self justify-self;
  • Units and Functions
    • fr - frakcija, nurodo kokia dali plokscio/aukscio turi priskirti
    • repeat(kartai, dydis) - nurodo kiek kartu naudoti tam tikra didi
    • minmax(maziausias dydis , didziausias dydis) - nurodo kiek vietos grido eilute ar stulpelis gali uzimti
    • fit-content()
    • min()
    • max()
    • subgrid
    • masonry - experimental
