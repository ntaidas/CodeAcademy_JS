Semantics and Page Layout

Kuriant internetine svetaine/aplikacija reiketu naudoti semantinius tagus, tam kad butu tvarkinga puslapio struktura ir joje programuojant butu lengviau susigaudyti.

  • HEADER - puslapio virsus skirtas logo, menu, login, sign In ir pns...
    • nav - navigacija, skirta menu
  • MAIN - pagrindinis turinys
    • section - puslapio dalis skirta aprasyti vienai temai
    • article - puslapio dalis skirta aprasyti vienai temai
    • figure - skirtas talpinti image, video, audio, map, iframe, gif...
    • aside - sonine juosta(dazniausia naudojama papildomam menu)
  • FOOTER - copyright, terms and use, cookies, privacy policy,partners, social links ir etc...

Box Model

Kiekvienas elementas turi savo box modeli, kuris susideda is zemiau isvardintu elementu, nuo vidaus i isore

  1. Content - turinys
  2. Padding - pamusalas
  3. Border - remelis
  4. Margin - parastes
  5. Position - poziciju atitraukimai
Box Model Pavyzdys

Padding

Padding nustato per koki atstuma turinys yra atitraukiamas nuo remelio.

  • padding-top: 10px;
  • padding-right: 10px;
  • padding-bottom: 10px;
  • padding-left: 10px;
  • padding: 5px(visi);
  • padding: 10px(virsus, apacia) 5px(sonai);
  • padding: 20px(virsus) 10px(sonai) 20px(apacia);
  • padding: 10px(virus) 20px(desine) 10px(apacia) 20px(kaire) ;

Margin

Marginas atitraukia elementa nuo aplinkiniu elementu.

  • margin-top: 10px;
  • margin-right: 10px;
  • margin-bottom: 10px;
  • margin-left: 10px;
  • margin: 5px(visi);
  • margin: 10px(virsus, apacia) 5px(sonai);
  • margin: 20px(virsus) 10px(sonai) 20px(apacia);
  • margin: 10px(virus) 20px(desine) 10px(apacia) 20px(kaire);
  • margin: 0 auto (centruoja); - turi buti blokinis ir tureti declaruota ploti.

Border

Remelis kazkokio tipo, storio ir spalvos juosta skirianti padding ir margin.

  • border:1px(storis) solid(stilius) red(spalva);
    • border-width:1px;
      • border-width: 10px 20px 30px; 40px
      • border-top-width:1px;
    • border-style:solid;
      • border-right-style:solid;
      • border-style:solid dotted riged dotted;
    • border-color:red;
      • border-right-color:red;
      • border-color:red blue green black;
  • border-radius: 5px(atstumas tarp krastu) gali but px arba procentai(max50%)
    • border-top-left-radius: 10px arba galima daryti 2 (10px 20px) ar daugiau, ar per ( / ) del idomesnes strukturos; Veikia panasiai kaip ir padding ir margin.

Selectors

  • Elementas/tag
  • Class
  • ID
  • selector1 selector2 - viduje, nesvarbu kaip giliai
  • selector1 > selector2 - elementas elemente, direct child. Nesibaigia ties vienu ir gali nestint kiek reikia.
  • element.class - tik elementas su ta klase
  • element#id - tik elementas su tik tuo id
  • selector, selector - nurodyti selectoriai gaus vienodus nustatymus. Galima visaip maisyti.
  • selector1 + selector2 - is karto sekantis elementas tame paciame lygmenyje
  • selector1 ~ selector2 - nesvarbu kaip toli einantiems tame paciame lygmenyje, uz elemento
  • selector:hover - pelei uzejus ant selectinto elemento
  • * - parenka viska
