Parent ir Child elementai

Child elementas turi viena vieninteli parent elementa(ta kurio viduje yra).

Tevinis elementas turi tureti bent viena elementa savo viduje.

Paveldimumas

Stiliai gali buti paveldimi is tevinio ar tevinio tevinio... elemento. Bet jie turi maziausia prioriteta, taigi jie gali buti labai lengvai overwritinti kitu stiliu. Stiliai kurie itakoja containerio turini YRA paveldimi, o stiliai kurie itakoja pati konteineri NERA paveldimi(ne 100%).

User Agent Stylesheet

Normalize or Reset CSS

Reset CSS - suvienodina/ panaikina visu tagu stilius. (not a good option)

Normalize CSS - suvienodina stilius tarp skirtingu narsykliu, sutvarko nenorimus narsykliu numatytuosius stilius ir pritaiko modernius stiliaus standartus.

CSS Priorities

  1. !important - ignoruoja visus prioritetus(neveikia su inherit)
  2. Inline Style - 1000
  3. ID - 100
  4. Class - 10
  5. Element - 1
  6. CSS order - jei specifikacija yra vienoda.
  7. User Agent Stylessheets - narsyklinis
  8. Inheriting

Esant dviem selektoriam, kurie nusako to paties elemento skirtingus stilius, yra atsizvelgiama i tu selektoriu specifikacijas. Aukstesni bala turintis selectorius turi aukstesni prioriteta.

Esant dviem vienodos specifikacijos selektoriams, keiciant ta pati stiliu, tuomet yra atsizvelgiama i CSS eiliskuma. Kuris selektorius yra aprasytas veliau, tas tures aukstesni prioriteta.

Jeigu imanoma - nenaudoti !important prieraso. Vienintelis budas overwritinti !important tai su aukstesnes specifikacijos ar eiliskumo kitu !important tagu.
